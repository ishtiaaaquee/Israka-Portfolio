## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. Or serve through XAMPP/Apache server

## Local Development

If using XAMPP:
1. Place the project in `htdocs` folder
2. Start Apache server
3. Navigate to `http://localhost/Israka-Portfolio`

## Customization

To personalize the portfolio:
- Update contact links in `index.html` (email, GitHub, LinkedIn)
- Add your own content in the About section
- Customize colors in `styles.css` (CSS variables in :root)
